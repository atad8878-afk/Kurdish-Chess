# ♔ Kurdish Chess ♚

A world-class chess game implementation featuring Kurdish language support, beautiful UI, and powerful AI opponents.

## 🎯 Features

- **Complete Chess Engine**: Full rule implementation with move validation
- **AI Opponent**: Multiple difficulty levels powered by minimax algorithm
- **Kurdish Interface**: Complete Kurdish language support for UI and documentation
- **Beautiful UI**: Modern, responsive interface for desktop and web
- **Game Analysis**: Move history, board evaluation, and hints
- **Multiplayer**: Local PvP gameplay
- **Performance**: Optimized for smooth gameplay on all devices

## 🚀 Getting Started

### Prerequisites
- Node.js 16+ or Python 3.8+
- Git

### Installation
```bash
git clone https://github.com/atad8878-afk/Kurdish-Chess.git
cd Kurdish-Chess
npm install  # or pip install -r requirements.txt
```

### Running the Game
```bash
npm start    # or python main.py
```

## 🎮 How to Play

1. Start a new game
2. Choose game mode (PvP, vs AI)
3. Select difficulty level (for AI)
4. Make moves by clicking source and destination squares
5. Win by checkmating your opponent's king

## 🤖 AI Levels

- **Beginner**: Random legal moves
- **Intermediate**: Basic tactical evaluation
- **Advanced**: Full minimax with alpha-beta pruning
- **Expert**: Advanced evaluation with opening/endgame knowledge
